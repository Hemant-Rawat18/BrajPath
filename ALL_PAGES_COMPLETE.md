### 2. Load Seed Data
```bash
cd backend
node src/seeds/seedData.js
```

This will populate:
- 9 temples (Mathura & Vrindavan)
- 7 hotels (different categories)
- 4 transport routes with prices

### 3. Get Mapbox Token
1. Sign up: https://account.mapbox.com/access-tokens/
2. Copy your token (starts with `pk.`)
3. Update `/frontend/.env.local`:
   ```bash
   NEXT_PUBLIC_MAPBOX_TOKEN=pk.your_actual_token_here
   ```
4. Restart frontend: `npm run dev`

### 4. Configure Environment Variables

**Backend** (`.env`):
```bash
PORT=3000
MONGO_URI=mongodb://localhost:27017/brajpath
JWT_SECRET=your_strong_secret_key_here
NODE_ENV=development
```

**Frontend** (`.env.local`):
```bash
NEXT_PUBLIC_API_URL=http://localhost:3000
NEXT_PUBLIC_MAPBOX_TOKEN=pk.your_token_here
NEXT_PUBLIC_APP_NAME=BrajPath
NEXT_PUBLIC_APP_URL=http://localhost:3001
```
